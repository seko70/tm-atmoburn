# AtmoBurn Services - Black Starmap Background
This is Tampermonkey (https://www.tampermonkey.net/) script for Atmoburn game (https://www.atmoburn.com/).

## What it does
Eliminates the empty space above the fleet and colony menus.
Idead and original code by Mario.

## How to install
- You should have Tampermonkey (https://www.tampermonkey.net/) or equivalent
- Open `abs-eliminate-empty-space-above-menus.user.js` file and go "Raw" in your browser - Tampermonkey should offer you "Install" button - and thats it.

## How to use
You should see (almost) none empty space over Colonies and Fleets side menus; useful if you have more colonies or fleets.
