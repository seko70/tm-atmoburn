# AtmoBurn Services - Black Starmap Background
This is Tampermonkey (https://www.tampermonkey.net/) script for Atmoburn game (https://www.atmoburn.com/).

## What it does
Changes background image to a black one in WarFacts/Atmoburn Starmap, for better readability
Original by guardian.

## How to install
- You should have Tampermonkey (https://www.tampermonkey.net/) or equivalent
- Open `abs-black-starmap-background.user.js` file and go "Raw" in your browser - Tampermonkey should offer you "Install" button - and thats it.

## How to use
Open 3D map and you should see tiny little stars more clearly. Or ships of your enemies hiding there.
